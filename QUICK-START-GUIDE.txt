═══════════════════════════════════════════════════════════════════
  PART 1: AUTOMATIC REORDER SYSTEM
═══════════════════════════════════════════════════════════════════

📍 URL: http://localhost:3000/dashboard/purchases/suggestions

✅ WHAT IT DOES:
- Analyzes sales from ALL locations (Main Warehouse + all branches)
- Calculates total stock across ALL locations
- Shows products that need reordering
- Prioritizes by urgency (Critical/High/Medium/Low)
- Generates Purchase Orders automatically

🎯 HOW TO USE:

Step 1: Open Suggestions Page
→ Navigate to: Dashboard → Purchases → Purchase Suggestions

Step 2: Review Products Needing Reorder
→ See urgency badges:
  🔴 CRITICAL (< 3 days stock) - Order immediately!
  🟠 HIGH (3-7 days) - Order soon
  🟡 MEDIUM (7-14 days) - Plan order
  🔵 LOW (> 14 days) - Monitor

Step 3: Handle Products Without Suppliers (Yellow Rows)
→ Yellow background = No supplier assigned
→ Click "Assign Supplier" button
→ Select supplier from dropdown
→ Product is now ready for PO! ✅

Step 4: Generate Purchase Orders
→ Check boxes for products to order
→ Click "Generate Purchase Orders" button
→ Choose delivery location (Main Warehouse)
→ Set expected delivery days
→ Done! POs created automatically (grouped by supplier) ✅

═══════════════════════════════════════════════════════════════════
  PART 2: PROFESSIONAL PURCHASE ORDERS
═══════════════════════════════════════════════════════════════════

📍 URL: http://localhost:3000/dashboard/purchases/[id]

✅ WHAT YOU GET:
- Professional PO with your company header
- Complete supplier information
- Clean table format
- Financial totals
- Signature lines
- Print-ready format

🎯 HOW TO USE:

Option A: PRINT TO PDF
1. Open any Purchase Order
2. Click "Print" button (printer icon)
3. In print dialog, select "Save as PDF"
4. Save to your computer
5. Done! Professional PO ready to send ✅

Option B: EMAIL TO SUPPLIER (ONE CLICK!)
1. Open Purchase Order
2. Look for green "Email to Supplier" button
   (Only shows if supplier has email)
3. Click the button
4. Review pre-filled subject and message (editable)
5. Click "Send Email"
6. Done! Supplier receives PDF automatically ✅

Option C: EXPORT TO EXCEL
1. Click "Export Excel" button
2. File downloads as .xlsx
3. Use for budget tracking/analysis ✅

  EXAMPLE WORKFLOW
═══════════════════════════════════════════════════════════════════

SCENARIO: Daily stock check and reordering

⏰ TIME: 10 minutes total

1️⃣ MORNING CHECK (2 min)
   → Open /dashboard/purchases/suggestions
   → Review critical and high-priority items
   → Note: 5 critical items, 8 high priority

2️⃣ HANDLE MISSING SUPPLIERS (1 min)
   → See 2 products in yellow
   → Click "Assign Supplier" on each
   → Select appropriate suppliers
   → Products now ready to order ✅

3️⃣ GENERATE PURCHASE ORDERS (2 min)
   → Select all 13 products (checkboxes)
   → Click "Generate Purchase Orders"
   → Choose "Main Warehouse" for delivery
   → Set "7 days" expected delivery
   → System creates 4 POs (one per supplier) ✅

4️⃣ EMAIL TO SUPPLIERS (5 min total)
   → Open first PO
   → Click "Email to Supplier" (green button)
   → Review/edit message if needed
   → Click "Send Email" ✅
   → Repeat for 3 other POs

5️⃣ DONE! 🎉
   → All 4 suppliers notified
   → Professional PDFs sent
   → Stock will arrive in 7 days
   → Total time: 10 minutes

═══════════════════════════════════════════════════════════════════
  TIPS & TRICKS
═══════════════════════════════════════════════════════════════════

💡 TIP 1: Focus on CRITICAL items first
   → Red badges mean you'll run out in < 3 days
   → Order these immediately!

💡 TIP 2: Assign suppliers as you go
   → Don't wait until you need to order
   → Fix yellow rows when you see them

💡 TIP 3: Use filters to prioritize
   → Filter by location to see specific branch needs
   → Filter by urgency to see critical items only
   → Filter by supplier to plan orders

💡 TIP 4: Keep supplier emails updated
   → Green email button only shows if email exists
   → Update supplier profiles to enable one-click emailing

💡 TIP 5: Print/save PDFs for records
   → Create digital filing system
   → Keep copies for audits
   → Track supplier performance
